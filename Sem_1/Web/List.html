Lists in HTML

Two types of lists in html:
Unodered list,Odered list

Unodered Lst

An unodered list starts with the 'ul' tag.Each list item is entered using 'li' tag the list is marked with bullets by default.

example
  • Coffee
  • Milk
  • tea

An odered list starts with the 'ol' tag.Each list item is entered using 'li' tag the list is marked with numbers by default,but it can be changed by using 'type' attribute

example
  1. Coffee
  2. Milk
  3. tea
